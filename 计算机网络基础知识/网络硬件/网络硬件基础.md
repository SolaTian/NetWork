# 1、网络通用基础知识

## 1.1、网络的构成要素

**<font size =4 color = red>`LAN`和`WAN`</font>**

|分类|说明|
|-|-|
|`LAN`（局域网）|大多数情况下，`LAN`可以使用以太网帧格式的以太网协议进行通信，网络中还有支持该标准的交换机和路由器。由路由器处理跨越异构子网和发送至互联网的通信。|
|`WAN`（广域网）和`MAN`（城域网）|地理位置相距较远的各点间建立起连接的计算机通信网络。如一个企业的总部与各个分部之间连接就会使用`WAN`，`WAN`的范围比`MAN`的范围更广|

可以将互联网理解成为全世界的`WAN`的互联。

**<font size =4 color = red>`OSI`参考模型</font>**

|层级|数据形式|主要协议|
|-|-|-|
|物理层|比特流|`FDDI`，`NRZ`等|
|数据链路层|帧|`ATM`，`PPP`，以太网|
|网络层|分组、数据报|`IP`，`IPX`|
|传输层|段，消息|`TCP`，`UDP`等|
|会话层|应用数据|`SSL`等|
|表示层|应用数据|`ASCII`，`UTF-8`等|
|应用层|应用数据|`HTTP`，`FTP`，`SMTP`等|

**<font size =4 color = red>`TCP/IP`层模型</font>**

|`TCP/IP`各层|`OSI`各层|地址|对应网络硬件|
|-|-|-|-|
|数据链路层|数据链路层+物理层|`MAC地址`|`L2交换机`，无线`LAN`接入点|
|网络层|网络层|`IP地址`|路由器、`L3交换机`|
|传输层|传输层|端口号|`L4交换机`、防火墙|
|应用层|会话层+表示层+应用层|根据应用程序的不同而不同|`L7交换机`、防火墙、代理|


## 1.2、`LAN`和以太网

以太网帧格式可以参考`图解TCP/IP`的数据链路层。

从千兆以太网开始，就逐步开始使用全双工通信，到了万兆以太网，就彻底舍弃了`CSMA/CD`这种半双工通信方式。

**<font size =4 color = red>以太网</font>**

以太网按照其发展历程，可以分为：
- 10Mbit/s以太网
- 快速以太网
- 千兆以太网
- 万兆以太网

以太网的传输采用`小端模式（LSB）`传输比特流，会从最低位开始传送。而`FDDI`以及令牌环等网络采用`大端模式`传送。

## 1.3、网络设备的构成要素

**<font size =4 color = red>存储器</font>**

|存储器的分类|说明|
|-|-|
|`RAM`|随机存取存储器，当关闭电源或者重新启动，将清空所有数据。|
|`ROM`|只读存储器，关闭电源时保存的数据不会丢失。|
|`NVRAM`|由于拥有非易失性，即使关闭电源或重新启动也不会丢失数据，一般用于保存设置文件。|
|闪存|具有电可擦可编程特性的只读存储器，用来保存操作系统以及设置文件|

|`RAM`种类|说明|
|-|-|
|`DRAM`|动态随机存储器|
|`SDRAM`|同步动态随机存储器|
|`SRAM`|静态随机存储器|

|闪存的种类|说明|
|-|-|
|`Flash EEPROM`|无需电源就能写入数据的非易失性存储器之一|
|`Compact Flash Card`|存储卡，内部使用了`Flash EEPROM`芯片|
|`USB存储器`|内部使用了`Flash EEPROM`芯片|
