# 10、构建`Web`内容的技术

## 10.1、`HTML`

**<font size =4 color = red>`HTML`</font>**

> `HTML`:超文本标记语言，为了发送`Web`上的超文本开发的标记语言。

`超文本`是一种文档系统，可以将文档中任意位置的信息与其他信息(文本或者图片等)建立关联，即`超链接文本`。

`标记语言`是指通过在文档的某部分穿插特别的字符串标签，用来修饰文档的语言，这些特殊的字符串叫做`HTML`标签。`HTML`经过浏览器的解析，渲染之后呈现出来的结果就是`Web`页面。

**<font size =4 color = red>`CSS`</font>**

`CSS`(层叠样式表)可以指定如何展现`HTML`内的各种元素，属于样式表标准之一，即使是相同的`HTML`，通过改变应用的`CSS`，用浏览器看到的页面外观也会随之改变。`CSS`的设计理念就是让文档的结构和设计分离。

**<font size =4 color = red>动态`HTML`</font>**

> 动态`HTML`：指使用客户端脚本语言将静态的HTML内容变成动态的技术的总称。

鼠标单击点开的新闻，谷歌地图的滚动的地图就用到了`HTML`。动态`HTML`技术就是通过调用客户端脚本语言`JavaScript`实现对`HTML`的`Web`页面的动态改造。利用`DOM`可以指定欲发生动态变化的`HTML`元素。`DOM`是一个应用编程接口。

## 10.2、`Web`应用 

> `Web`应用：指通过`Web`功能提供的应用程序。如购物网站，网上银行，搜索引擎等。

服务端在接收到客户端的请求之后返回由程序创建的`HTML`内容，这种内容叫做动态内容，事先编写好的`HTML`内容叫做静态内容。`Web`应用作用于动态内容之上。


**<font size =4 color = red>`CGI`</font>**

> `CGI`是指`Web`服务器在接收到客户端发过来的请求后转发给程序的一组机制。

在`CGI`的作用下，程序会对请求内容做出相应的动作，比如创建`HTML`等动态内容。但是有个弊端，就是每次请求都要启动新的`CGI`程序，一旦访问量过大，`Web`服务器就会承受相当大的负载，使用`Java`实现的`Servlet`，它运行在`Web`服务器相同的进程中，收到的负载较小。

使用`CGI`的程序叫做`CGI`程序，通常是`Perl`，`PHP`，`Ruby`，`C`等编写。

## 10.3、数据发布的格式和语言

> `XML`：可扩展标记语言。和`HTML`相比，它对数据的记录方式做了特殊处理。

`XML`使得互联网数据共享变得更容易。`HTML`不适用于记录数据结构。从`XML`中读取数据比`HTML`中简单。

>`JSON`:一种以`JavaScript`的对象表示法为基础的轻量级数据标记语言，能够处理`false`/`null`/`true`/对象/数组/数字/字符串。
